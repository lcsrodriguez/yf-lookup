## Getting started

```
git clone https://github.com/lcsrodriguez/yf-finance.git
cd yf-finance/

python3 -m venv .venv
source .venv/bin/activate

pip3 install -r requirements.txt
pip3 freeze
```


```
uvicorn api:app --reload
```
